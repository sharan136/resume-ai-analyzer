# 🧠 Resume AI Analyzer

An AI-powered web application that analyzes resumes, extracts key information, evaluates ATS (Applicant Tracking System) compatibility, matches skills with job requirements, and provides personalized improvement suggestions.

The project aims to help job seekers improve their resumes and increase their chances of passing automated resume screening systems.

---

## 🚀 Features

-  Upload resume files
-  Extract important resume information
-  Analyze ATS compatibility score
-  Match resume skills with job requirements
-  Provide AI-based resume improvement suggestions
-  Simple and user-friendly web interface
-  Identify strengths and areas for improvement

---

## ⚙️ Installation & Setup

Follow these steps to run the project locally.

### Step 1: Clone the Repository

Open Command Prompt/Terminal and run:

```bash
git clone https://github.com/sharan136/Resume-AI-Analyzer.git
```

---

### Step 2: Navigate to the Project Directory

```bash
cd Resume-AI-Analyzer
```

---

### Step 3: Install Required Dependencies

```bash
pip install -r requirements.txt
```

---

### Step 4: Run the Application

```bash
python app.py
```

---

### Step 5: Open the Website

Open your browser and visit:

```
http://127.0.0.1:5000/
```


## 📦 Requirements

- Python 3.x
- Flask
- PyPDF2
- ReportLab


## 🔄 Application Workflow

1. User uploads a resume file.
2. The application extracts text and important details from the resume.
3. Resume content is analyzed using text-processing techniques.
4. ATS compatibility and skill matching are evaluated.
5. The system provides suggestions to improve the resume.
